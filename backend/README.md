# Telegram mini app backend

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript project for Telegram mini apps.

## Project setup

In the root of the workspace, run:

```bash
$ yarn install
```

## Compile and run the project

```bash
# development
$ yarn dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
